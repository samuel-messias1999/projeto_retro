INFORMAÇÕES

Objetivo do Projeto:

O objetivo do projeto é de ser um desafio para os integrantes do grupo, para a realização dele é necessário a utilização de diversas linguagens de programação de diferentes áreas de desenvolvimento (tanto de Front-End na elaboração do site, Back-End para a criação e conexão com o banco de dados, e de Desenvolvimento de Jogos Digitais para a criação do jogo para navegador) bem como a utilização de frameworks para o gerenciamento do projeto, viabilizando assim o término dele dentro do prazo de entrega.

Descrição do Projeto:

O projeto tem como base a criação de um site utilizando HTML, CSS e Javascript, e conecta-lo utilizando PHP à um banco MySQL para controlar o acesso ao jogo para navegador (este feito utilizando o motor gráfico Godot e sua linguagem proprietária GDScript que contém sintaxe baseada em Python).

Além dessa base, também se foi utilizado a metodologia ágil de gerenciamento de projetos Kanban (por meio do Trello), o framework Front-End Bootstrap, e diversas bibliotecas Front-End (Jquery, Sass, Swiper, GLightbox, AOS, Boxicons e Remix Icon).
